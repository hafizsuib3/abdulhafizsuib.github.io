# Abdul Hafiz Suib Portfolio

This repository now uses a custom GitHub Pages portfolio website.

## Files

- `index.html` — main portfolio webpage
- `style.css` — website styling
- `script.js` — simple interaction
- `profile1.jpg` — profile image already in this repository

## Main Feature

The Projects section uses a **Project Pipeline Overview** to show which parts of the data lifecycle each project covers:

1. Data Source
2. Storage
3. Cleaning
4. ML / AI
5. Visualization
6. Deployment

Each project row is clickable and jumps to the full project detail section.
